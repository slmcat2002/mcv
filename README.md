# Usage:

### Execute:
```
sudo ./build.sh [port] [password] [encrypt]
```

### Example:
```
./build.sh 404 MiaoMiaoMia0? camellia-256-cfb
```

### Ecrypt Recommended: 
- camellia-128-cfb
- camellia-192-cfb
- camellia-256-cfb
- chacha20-ietf-poly1305
- chacha20-ietf
